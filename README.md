#  Rust Voice Assistant with MathJax Support

Голосовой AI-ассистент на Rust с поддержкой математических формул и химических уравнений.

##  Особенности

-  **Голосовое управление** - распознавание речи через Groq Whisper
-  **Математические формулы** - красивое отображение через MathJax
-  **Химические уравнения** - поддержка молекул и реакций
-  **Веб-интерфейс** - современный дизайн с градиентами
-  **Контекстный диалог** - ИИ помнит историю разговора
-  **Real-time** - мгновенные ответы через WebSocket

##  Примеры формул

### Математика:
- Квадратное уравнение: $ax^2 + bx + c = 0$
- Решение: $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$
- Интеграл: $\int_{a}^{b} f(x) dx$
- Производная: $\frac{d}{dx}f(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$

### Химия:
- Молекулы: H₂O, CO₂, C₆H₁₂O₆
- Ионы: Ca²⁺, SO₄²⁻, NH₄⁺
- Реакции: 2H₂ + O₂ → 2H₂O
- Комплексы: [Cu(NH₃)₄]²⁺

##  Технологии

- **Backend**: Rust + Axum + WebSocket
- **AI**: Groq API (Whisper + Llama)
- **Frontend**: HTML5 + CSS3 + JavaScript
- **Math**: MathJax 3.0
- **Audio**: Web Audio API

##  Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ВАШ_USERNAME/rust-voice-assistant-math.git
cd rust-voice-assistant-math
```

2. Установите зависимости:
```bash
cargo build
```

3. Создайте `.env` файл:
```bash
cp .env.example .env
```

4. Добавьте ваш Groq API ключ в `.env`:
```
GROQ_API_KEY=your_groq_api_key_here
```

5. Запустите сервер:
```bash
cargo run
```

6. Откройте браузер: http://localhost:3000

##  Использование

1. **Голосовой ввод**: Нажмите и удерживайте кнопку записи
2. **Текстовый ввод**: Используйте вкладку "Текст"
3. **Тестирование**: Нажмите кнопки "Тест математики" или "Тест химии"

##  Настройка

### Переменные окружения:
- `GROQ_API_KEY` - ключ API для Groq
- `PORT` - порт сервера (по умолчанию 3000)

### Системный промпт:
Отредактируйте `system_prompt.txt` для настройки поведения ИИ.

##  API

### WebSocket эндпоинты:
- `/ws` - основное WebSocket соединение

### Поддерживаемые сообщения:
- `text:ваш_текст` - текстовый запрос
- `morse:код_морзе` - декодирование азбуки Морзе
- `ping` - проверка соединения
- `clear_context` - очистка контекста
