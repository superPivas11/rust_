ИНСТРУКЦИЯ ПО УСТАНОВКЕ БИБЛИОТЕК ДЛЯ ESP32
═══════════════════════════════════════════════════════════════

НУЖНЫЕ БИБЛИОТЕКИ:
1. WiFi (встроенная в ESP32)
2. WebSocketsClient
3. U8g2
4. EEPROM (встроенная в ESP32)
5. Wire (встроенная в ESP32)

═══════════════════════════════════════════════════════════════
КАК УСТАНОВИТЬ В ARDUINO IDE:
═══════════════════════════════════════════════════════════════

1. Открой Arduino IDE

2. Меню: Sketch → Include Library → Manage Libraries...
   (или Ctrl+Shift+I)

3. В поисковой строке введи и установи:

   ┌─────────────────────────────────────────────────────────┐
   │ WebSockets by Markus Sattler                            │
   │ Версия: 2.4.0 или новее                                 │
   │ Нажми "Install"                                         │
   └─────────────────────────────────────────────────────────┘

   ┌─────────────────────────────────────────────────────────┐
   │ U8g2 by oliver                                          │
   │ Версия: 2.35.0 или новее                                │
   │ Нажми "Install"                                         │
   │ (Если спросит про U8g2lib - тоже установи)             │
   └─────────────────────────────────────────────────────────┘

4. Закрой Library Manager

═══════════════════════════════════════════════════════════════
ПРОВЕРКА УСТАНОВКИ:
═══════════════════════════════════════════════════════════════

1. Меню: Sketch → Include Library
2. В списке должны быть:
   ✓ WebSockets
   ✓ U8g2

═══════════════════════════════════════════════════════════════
КАК ПРОШИТЬ:
═══════════════════════════════════════════════════════════════

1. Открой файл ESP32_COMPLETE_FINAL.ino в Arduino IDE

2. Убедись что у тебя есть файлы анимаций:
   - animation_data_1.h
   - animation_data_2.h
   - animation_data_3.h
   (Они должны быть в той же папке)

3. Выбери плату:
   Tools → Board → ESP32 Arduino → ESP32 Dev Module

4. Выбери порт:
   Tools → Port → COM# (твой ESP32)

5. Нажми Upload (стрелка вправо)

6. Дождись "Done uploading"

ЕСЛИ ОШИБКИ ПРИ КОМПИЛЯЦИИ:
═══════════════════════════════════════════════════════════════

Ошибка: "WebSocketsClient.h: No such file"
Решение: Установи библиотеку WebSockets

Ошибка: "U8g2lib.h: No such file"
Решение: Установи библиотеку U8g2

Ошибка: "animation_data_1.h: No such file"
Решение: Скопируй файлы анимаций в папку с .ino файлом
